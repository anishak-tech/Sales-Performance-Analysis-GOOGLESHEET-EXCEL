# Sales-Performance-Analysis-GOOGLESHEET-EXCEL
## 🔹 Project Overview

This project analyzes transactional sales data to evaluate regional performance, product demand, and salesperson contribution.

The goal was to extract actionable business insights using structured data cleaning and Pivot Table analysis in Microsoft Excel / Google Sheets.

---

## 🔹 Business Objectives

* Identify the highest revenue-generating region
* Evaluate individual salesperson performance
* Analyze product-wise revenue contribution
* Measure Average Order Value (AOV)
* Identify regional product demand trends

---

## 🔹 Dataset Description

The dataset contains the following fields:

* Order ID
* Date
* Region
* Salesperson
* Product
* Quantity
* Unit Price
* Total Sales

Total Records Analyzed: 40

---

## 🔹 Data Cleaning & Preparation

Before analysis, the following preprocessing steps were performed:

* Standardized inconsistent region names (e.g., “west” → “West”)
* Corrected capitalization inconsistencies in salesperson names
* Reviewed duplicate Order IDs
* Standardized mixed date formats
* Validated Total Sales (Quantity × Unit Price)

Data cleaning ensured accuracy and reliability of results.

## 🔹 Key Findings

### 1️⃣ Regional Performance

The South region generated the highest total revenue (₹ 972,000), indicating strong market performance.

### 2️⃣ Salesperson Contribution

Anil was the top-performing salesperson with ₹ 844,000 in total sales.

### 3️⃣ Product Performance

Laptop was the highest revenue-generating product category (₹ 1,287,000).

### 4️⃣ Regional Product Preference

* North → Laptop
* South → Tablet
* East → Tablet
* West → Laptop

Product demand varies across regions, suggesting opportunities for targeted marketing strategies.
